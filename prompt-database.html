📐 Prompt Record Schema
Every prompt in the database conforms to this schema. External tools consuming the exported JSON or API endpoint should use these field definitions. Variables in prompt_text use {{double_brace}} syntax (Mustache-compatible).
meta — Identity & Metadata
idstringUnique slug, e.g. prompt_hr_001REQ
titlestringHuman-readable nameREQ
teamenumHR | Engineering | Product | Sales | Marketing | Operations | LeadershipREQ
statusenumdraft | review | published | archivedREQ
complexityenumBeginner | Intermediate | AdvancedREQ
versionstringSemver-style, e.g. v1.0, v2.3
ownerstringTeam or person responsible
ai_toolstringTarget AI tool or "Any"
created_atISO dateAuto-set on creation
updated_atISO dateAuto-updated on every save
content — Prompt Body
use_casestringProblem solved, when to useREQ
system_contextstringOptional system prompt / persona
prompt_textstringFull prompt; use {{var}} for inputsREQ
inputsarrayArray of variable definitions (see below)
expected_outputstringDescription of correct output
example_outputstringAnonymized sample output
inputs[] — Variable Definition
keystringMachine key matching {{key}} in promptREQ
labelstringHuman-readable display nameREQ
descriptionstringGuidance on what to enter
requiredbooleanWhether value is mandatory
examplestringPlaceholder / example value
taxonomy & stats
tagsstring[]Lowercase, hyphen-separated keywords
stats.viewsnumberTotal view count
stats.copiesnumberNumber of times prompt was copied
stats.likesnumberCommunity like count

Export Full Database — JSON
Download all published prompts as a structured JSON array. Use this as a static data source for internal tools, SharePoint lists, or CI pipelines.

Flat CSV export for Excel, Power BI, or SharePoint list import. Each row is one prompt; inputs are serialized as a delimited string.

Select a prompt in the Database tab, then copy its full JSON record for use in API calls, Power Automate flows, or webhook payloads.

CSV format is compatible with SharePoint list import. Map columns to your custom list fields and use Power Automate to keep it in sync.
