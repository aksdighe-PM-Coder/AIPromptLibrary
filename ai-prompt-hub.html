🏛 Governance Model

The AI Prompt Hub is governed by a three-tier role model to ensure quality, accuracy, and continual improvement of published prompts.

Roles & Responsibilities
✍️
Contributor
Any employee
Team members who discover, test, and submit high-quality prompts for their team's use cases.
Submit new prompts via the form
Rate and comment on prompts
Flag outdated or incorrect prompts
Suggest edits via comments
🔍

Team leads & AI Champions

Approve or reject submissions
Edit prompt text and metadata
Request revisions from contributors
Assign tags and categories

✅ Prompt Writing Best Practices
✅ Do's
Use clear, specific instructions — tell the AI exactly what you want
Define the output format (e.g., bullet list, table, paragraph)
Use placeholders like [employee name] for variable inputs
Include tone/style guidance (e.g., "professional", "concise")
Test your prompt with at least 3 different inputs before submitting
Add an example output to help users know what to expect
Version your prompt when making significant changes
❌ Don'ts
Don't include real personal data or PII in prompts or examples
Don't use vague instructions like "write something about X"
Don't share prompts that include confidential company info
Don't assume the AI knows your internal jargon without context
Don't submit untested prompts — validate outputs first
Don't use prompts to generate content outside approved AI tools
🏷 Naming Conventions
Title Format
[Action] + [Output] + [Context]
e.g. "Generate Performance Review Summary"
Tag Format
Lowercase, hyphen-separated
e.g. performance-review, api-docs
Version Format
v1.0 → v1.1 (minor edit)
v1.0 → v2.0 (prompt rewrite)
